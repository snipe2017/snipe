C - Language
C is a high-level & general-purpose programming language that is ideal for developing firmware or portable applications.
JAVA
Java is a general-purpose OOPS language designed to be used in conjunction with the Java Virtual Machine.
C ++
C++ is an OOP language that is viewed by many as the best language for creating large-scale applications.

HTML 5
HTML is an acronym which stands for Hyper Text Markup Language. We can create static website by HTML only.
CSS 3
CSS stands for Cascading Style Sheet.CSS is used to design HTML tags & helps the web designers to apply style on HTML tags.
BOOTSTRAP
BOOTSTRAP is a front-end framework used for faster web development. It facilitates you to create responsive designs.

JQUERY
JQuery is a easy, light-weight & JavaScript lib. Make the things like HTML document traversal, animation effects to web page across all the major browsers.
JavaScript
JavaScript is an object-based scripting language that is lightweight and cross-platform. JavaScript is used to create interactive websites.
TypeScript
TypeScript is JavaScript for application-scale development. TypeScript is a strongly typed, object oriented, compiled language.

SQL
SQL stands for Structured Query Language. A database language for database creation, deletion, fetching, modifying rows.
PHP
PHP stands for HyperText Preprocessor. It is a open source, interpreted & object-oriented scripting language.
PYTHON
Python is an object-oriented, high level language, interpreted, dynamic & multipurpose programming language.
